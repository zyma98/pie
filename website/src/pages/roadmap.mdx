## Released in 0.2.0 (2026-01-20)

- **Apple Metal support** <span className="r-tag r-runtime">runtime</span>  
  Enable GPU execution on Apple silicon via Metal for laptop development and demos.

- **Multi-GPU support with mixed DP and TP** <span className="r-tag r-perf">performance</span>  
  Scale model size and throughput across devices using Data Parallelism and Tensor Parallelism.

- **Quantization support (float8/int8/float4)** <span className="r-tag r-perf">performance</span>  
  Reduce VRAM while preserving quality; per‑layer policies.

- **New model support** <span className="r-tag r-runtime">runtime</span>  
  Support for gpt-oss, ministral3, gemma2, gemma3, olmo3.

---

## In progress

- **PEFT adapters API (e.g., LoRA)** <span className="r-tag r-api">api</span>  
  Hot‑swap adapters per request/session with safe isolation.

- **KV‑cache quantization** <span className="r-tag r-perf">performance</span>  
  Larger batches and longer contexts via cache compression.

- **Inferlet standard libraries (Python / JS)** <span className="r-tag r-tooling">tooling</span>  
  Ergonomic helpers for common patterns; batteries included.


- **Inferlet package manager** <span className="r-tag r-tooling">tooling</span>  
  Discover, version, and share inferlets and libraries.

---
